<p style="padding:0;margin:0;font-size:.9em;" class="spsn-sponsor-heading"><em style="color:green;">Automattic&rsquo;s Ban on Passively Loaded Sponsorship Messages &raquo;</em></p>
<p style="padding:.2em;margin:0;" class="spsn-sponsor-text">Automattic Inc. has served notice that plugins which attempt to generate a tiny income for developers by passively loading sponsorship messages in the manner facilitated by PluginSponsors.com code will be expelled from the WordPress plugin repository. In the face of Automattic&rsquo;s threats against participating plugin authors, development of this plugin &mdash; together with many other plugins jointly representing 2.3 million aggregate downloads as of November 2011 &mdash; will no longer be supported by sponsorship messages. See: <a href="http://gregsplugins.com/lib/2011/11/26/automattic-bullies/">Automattic Bullies Plugin Developers &mdash; Again</a>.
